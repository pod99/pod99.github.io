# Летающие иконки — минимальный одностраничный сайт

Минималистичный статический проект с летающими иконками, всплывающими карточками и интеграциями (погода, OpenRouter AI, прогресс года).

## Быстрый старт

Откройте `index.html` в браузере. Проект не требует сборки.

## Структура

- `index.html` — главная страница.
- `css/style.css` — стили, адаптивные размеры иконок.
- `js/main.js` — логика анимаций, попапов и API.
- `data/content.json` — тексты/ссылки и API‑ключ.
- `assets/` — изображения и фавикон.

## Настройки

Все тексты и ссылки живут в `data/content.json`:

- `openrouterApiKey` — ключ OpenRouter (используется в попапе AI).
- `icons.*.title` — заголовок попапа.
- `icons.*.text` — текст попапа.
- `icons.*.link` — ссылка (если нужна кнопка-иконка).

## Что умеет

- Летающие иконки с физикой отскока от стен и друг друга.
- Попапы по клику на иконку.
- Погода в Москве (Open‑Meteo, без ключа).
- AI‑чат через OpenRouter API.
- Таймлайн как прогресс‑бар по прошедшим дням года.

## Фавикон

Фавикон берётся из `assets/icon.jpeg` и подключён в `index.html`.

## Примечания

- Проект полностью статический, без backend.
- API‑ключ OpenRouter хранится в `data/content.json`.
